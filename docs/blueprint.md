# **App Name**: ProductViewer

## Core Features:

- Sticky Mobile Header: Implement the sticky top navigation bar with system status, search bar, and action icons (share, cart, more options) for consistent access.
- Product Image Carousel: Display multiple product images in a scrollable carousel with a current image indicator (e.g., '1/8').
- Promotional Badges: Show various promotional badges (e.g., 'XTRA Gratis Ongkir', 'EXTRA 10%+ off', 'Hemat pakai Bonus') overlaying the product image.
- Product Pricing & Offers: Display the product's discounted price, original price, discount percentage, and special financing/customer offers clearly.
- Product Details Section: Show the product title and a brief description in a scrollable content area.
- Sticky Mobile Footer: Implement the sticky bottom navigation with 'Shop', 'Chat' links and a prominent 'Add to Cart' and 'Buy now' buttons.

## Style Guidelines:

- Light color scheme, reflecting the clean e-commerce UI in the screenshot.
- Primary brand color: Energetic Orange (#FF6600). This vibrant hue evokes excitement and urgency, perfect for calls to action.
- Background color: Soft Alabaster (#FDF8F5). A very light, subtly warm off-white provides a clean and inviting canvas for product display.
- Accent color: Deep Scarlet (#CC0F2F). This rich red, analogous to the primary and consistent with the discount elements, highlights key promotions and calls attention to important details.
- Body and headline font: 'Inter' (sans-serif) for its modern, neutral, and highly readable characteristics, suitable for various text elements in an e-commerce setting.
- Use outline-style icons with clear visual cues, consistent with contemporary mobile application design principles, for navigation and actions.
- The layout features sticky header and footer elements that persist on screen, providing constant access to critical navigation and purchasing options, while the main product content scrolls.
- Subtle slide transitions for the product image carousel, enhancing user experience without distracting from content.
